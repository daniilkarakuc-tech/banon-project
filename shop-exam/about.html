Про інтернет-магазин EcoHome

EcoHome — це навчальний інтернет-магазин побутових товарів, створений з метою демонстрації можливостей HTML, CSS та JavaScript.

На сайті представлений каталог товарів для дому, прибирання та кухні. Користувач може додавати товари до кошика та переглядати загальну суму замовлення.

Усі інтерактивні елементи реалізовані за допомогою JavaScript без використання серверних технологій.
